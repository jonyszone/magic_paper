# 🔮 Magic Paper

A magical mind-reading app inspired by the classic Nokia Symbian app. Think of a number, follow the steps, and watch as Magic Paper reveals your symbol!

## Features

- ✨ Beautiful dark mystical UI with animations
- 🔮 Animated crystal ball with glowing effects
- 📊 Stats tracking (play count, success rate)
- 🎯 Smooth page transitions
- 🌟 Floating particle background
- 📱 Works on iOS, Android, Web, and Desktop

## How It Works

1. Think of any two-digit number (10-99)
2. Add both digits together
3. Subtract the sum from your original number
4. Find your number in the grid and remember the symbol
5. Magic Paper will reveal the symbol you're thinking of!

## Getting Started

```bash
flutter pub get
flutter run
```

## The Secret

The math always results in a multiple of 9 (9, 18, 27, 36, 45, 54, 63, 72, 81). All multiples of 9 share the same symbol, making the prediction always correct!
